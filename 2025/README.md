# Advent of Code 2025

This year, all Julia again

## Setup

1. Install Julia [https://julialang.org/downloads/](https://julialang.org/downloads/)
2. Create `.env` file and get session cookie
3. Run with `julia --project=. script.jl` (or `julia. script.jl` if alias added to .bashrc)

To activate the environment, open Julia terminal, press `]` and run `activate .`.

To install all packages in the environment, open Julia terminal, press `]` and run `instantiate`.
